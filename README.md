## 🧠 Stock Price Predictor — LSTM Deep Learning

[![Repo](https://img.shields.io/badge/GitHub-Stock--Price--Predictor-181717?style=flat-square&logo=github)](https://github.com/Anandsavran/Stock-Price-Predictor-LSTM)

4-layer stacked LSTM trained on **17 years of POWERGRID.NS data** (4,208 trading days). Final training loss: **0.0021**.

**Model Architecture**

```python
model = Sequential([
    LSTM(50,  activation='relu', return_sequences=True, input_shape=(100, 1)),
    Dropout(0.2),
    LSTM(60,  activation='relu', return_sequences=True),
    Dropout(0.3),
    LSTM(80,  activation='relu', return_sequences=True),
    Dropout(0.4),
    LSTM(120, activation='relu'),
    Dropout(0.5),
    Dense(1)
])
# Parameters: 178,761 | Optimizer: Adam | Epochs: 50 | Final Loss: 0.0021
```

**Pipeline:** `yfinance` → EDA → MinMaxScaler → 100-day sliding window → 4-Layer LSTM → Inverse scale → ₹ predictions

**Stack:** `Python` `TensorFlow/Keras` `LSTM` `yfinance` `Pandas` `NumPy` `Scikit-learn` `Plotly` `Matplotlib`

## 🧬 NeuroLoom — Brainwave-to-Image AI *(Patent Filed)*

[![Repo](https://img.shields.io/badge/GitHub-NeuroLoom-181717?style=flat-square&logo=github)](https://github.com/Anandsavran/NeuroLoom)
[![Patent](https://img.shields.io/badge/Patent-Complete%20Filing%20%28PCT%20Recommended%29-8A2BE2?style=flat-square)]()

AI system that reads real-time EEG brainwave signals and converts them into dynamic generative artwork using GANs and Neural Style Transfer.

**Brainwave → Art Mapping**

| Brainwave | State | Visual Style |
|:---|:---|:---|
| Delta (0.5–4 Hz) | Deep sleep | Dark purples · Slow, heavy strokes |
| Theta (4–8 Hz) | Meditation / Creative | Indigo, violet · Dreamy, abstract |
| Alpha (8–14 Hz) | Relaxed / Calm | Soft blues, greens · Fluid, smooth |
| Beta (14–30 Hz) | Focused / Alert | Reds, yellows · Sharp, angular |
| Gamma (30–100 Hz) | High excitement | Bright white, orange · Chaotic, explosive |

**Pipeline:** EEG Headset → FFT/ICA Signal Processing → Cognitive Classifier (SVM/DNN) → GAN + NST Art Engine → Screen / VR / AR Output

**Stack:** `Python` `TensorFlow` `GANs` `Neural Style Transfer` `FFT` `Wavelet Transform` `SVM` `OpenBCI SDK` `OpenCV`

## 🔒 Deadlock Detection — Resource Allocation Graph Simulator

[![Repo](https://img.shields.io/badge/GitHub-Deadlock--Detection--RAG-181717?style=flat-square&logo=github)](https://github.com/Anandsavran/Deadlock-Detection-RAG)

Interactive Python GUI that visualizes Resource Allocation Graphs in real-time and detects deadlocks using NetworkX cycle detection.

```python
def detect_deadlock(self):
    cycles = list(nx.simple_cycles(self.graph))
    if cycles:
        messagebox.showwarning("Deadlock Detected!", f"Cycle found: {cycles}")
    else:
        messagebox.showinfo("Safe State", "No deadlock detected ✅")
```
