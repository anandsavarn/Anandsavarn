### ⬡ 03 — TRADING SIGNAL ENGINE · Algorithmic Market Intelligence

<div align="center">

[![Repo](https://img.shields.io/badge/⌥_GITHUB-Stock--Price--Predictor-181717?style=flat-square&logo=github&logoColor=white)](https://github.com/Anandsavran/Stock-Price-Predictor-LSTM)

</div>

> 4-layer stacked LSTM trained on 17 years of POWERGRID.NS market data (4,208 trading days) — generating closing price forecasts with technical indicator overlays and backtestable signal output.

```
DATA PIPELINE
─────────────────────────────────────────────────────────────────
yfinance API  →  4,208 OHLCV rows  →  EDA  →  MinMaxScaler [0,1]
     ↓
Sliding Window: 100-day look-back
x_train shape: (2845, 100, 1)
     ↓
4-Layer LSTM  ──  50 Epochs  ──  Adam Optimizer  ──  MSE: 0.0021
     ↓
Inverse Scale  →  ₹ Predictions  →  Actual vs Predicted chart
```

```python
model = Sequential([
    LSTM(50,  activation='relu', return_sequences=True, input_shape=(100,1)),
    Dropout(0.2),
    LSTM(60,  activation='relu', return_sequences=True),
    Dropout(0.3),
    LSTM(80,  activation='relu', return_sequences=True),
    Dropout(0.4),
    LSTM(120, activation='relu'),
    Dropout(0.5),
    Dense(1)
])
# Parameters: 178,761 · Optimizer: Adam · Epochs: 50 · Final Loss: 0.0021
```

**Signal Indicators Generated:** `Candlestick OHLC` · `MA100 / MA200` · `EMA100 / EMA200` · `RSI` · `MACD` · `Bollinger Bands` · `Actual vs Predicted`
