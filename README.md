### рҹ’Ў What Is a Deadlock?

```
In OS: Process A holds R1 and waits for R2
       Process B holds R2 and waits for R1
       вҶ’ Neither can proceed вҶ’ DEADLOCK

Visualized as a cycle in the RAG:

P1 в”Җв”Җrequestsв”Җв”Җв–ә R1 в”Җв”Җallocatedв”Җв”Җв–ә P2
в–І                                    в”Ӯ

```
